Brave Detection Block

Version:

While the extension is enabled:

window.navigator.brave

is removed

Notes

This extension's primary function is to block websites to detect you run Brave browser.

It only blocks the most common detection method which is checking the 
window.navigator.brave object
, and will potentially allow you to use websites that obstruct the Brave browser from accessing the page.

The extension does not touch the 
User-Agent
or try to prevent other methods of detecting Brave browser.

Additional steps one may take to improve privacy:
  • Turn off Brave crypto wallets that also allow websites to detect you are using Brave browser.
  • Use an User Agent randomizer
  • Turn off Brave Shields ( can be used to track Ad Blocker)
  • Use a trusted no Logs VPN
  • Use a new private browser session, which no previous store data
  • Have as few extensions as possible as some may leave traces and increase fingerprint surface
  • Use only open source extensions and inspect the source
  • In critical cases you can also double up the traffic through TOR
  • Only use a desktop environment that is secure and that you trust
